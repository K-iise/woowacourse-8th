# 미션 - 숫자 야구 게임

## 🚀 기능 요구사항

기본적으로 1부터 9까지 서로 다른 수로 이루어진 3자리의 수를 맞추는 게임이다.

1. 게임 초기화
- 상대방(컴퓨터)는 1에서 9까지 서로 다른 임의의 수를 3개 생성한다.
- 생성된 숫자는 순서를 가지고 컴퓨터는 숫자를 게임 시작 시 1번만 생성한다.

2. 사용자 입력
- 플레이어는 컴퓨터가 생각하고 있는 3개의 숫자를 입력한다.
- 입력은 3자리 수로 받는다

3. 입력 검증
- 입력된 길이가 3인지 검증한다
- 모든 문자가 숫자인지 검증한다.
- 숫자의 범위가 1~9 인지 검증한다.
- 서로 다른 임의의 수인지 검증한다.
- 검증 실패 시 IllegalArgumentException 에러를 발생시키고 애플리케이션을 종료한다.

4. 숫자 비교
- 컴퓨터의 수와 플레이어의 수가 같은 자리에 있으면 스트라이크이다.
- 다른 자리에 있으면 볼이다.
- 같은 수가 전혀 없으면 포볼 또는 낫싱이다.

5. 결과 출력
- 컴퓨터는 입력한 숫자에 대한 결과를 출력한다.
- 플레이어는 3개의 숫자를 모두 맞히면 게임이 종료된다.

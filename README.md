# 자동차 경주

### 기능 목록 정리

**1. 입력**
- [ ] 사용자로 부터 자동차 이름을 받는다.
  - 쉼표```'```를 기준으로 이름을 분리한다.
  - 이름이 5자 초과인 경우 ```IllegalArgumentException```를 발생시킨다.
  - 이름이 비어 있거나 중복된 경우 ```IllegalArgumentException```를 발생시킨다.
- [ ] 사용자로 부터 시도할 횟수를 입력 받는다.

**2. 게임 진행**
- [ ] 각 시도마다 모든 자동차가 랜덤 값(0-9)를 생성한다.
- [ ] 랜덤 값이 4 이상이면 전진(```-```) 가중치를 증가시킨다..
- [ ] 매 시도마다 각 자동차의 진행 상태를 출력한다.

**3. 우승자 선정**
- [ ] 가장 많이 전진한 자동차를 찾는다.
- [ ] 여러 대가 동점인 경우, 모든 우승자를 출력한다.

**4. 출력**
- [ ] 각 시도 후 ```자동차 이름 : -``` 형식으로 진행 결과를 출력한다.
- [ ] 경주가 끝난 후 ```"최종 우승자 : 자동차 이름```을 출력한다.

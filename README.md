# java-planetlotto-precourse

## 기능 목록 

**로또 구입**
- 로또 1장의 가격은 500원이고, 구입 수량을 계산한다.
- 남은 금액이 존재할 경우 `IllegalArgumentException`를 발생시킨다.

**로또 생성**
- 로또 번호의 범위는 1 ~ 30 까지 이며, 중복되지 않는 숫자 6개로 구성된다.
- 생성된 로또 번호는 오름차순으로 정렬된다.

**로또 발행**
- 로또 구입 정보의 요청으로 로또를 발행한다.
- 로또 구입 수량 만큼 로또 생성을 호출한다.

**당첨 번호 생성**
- 당첨 번화 보너스 번호의 중복 여부를 검사한다.
- 당첨 번호를 오름차순으로 정렬한다.

**당첨 조회**
- 당첨 번호와 로또 번호를 비교하여 등수를 결정한다.

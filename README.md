# java-planetlotto-precourse

## 기능 목록 

**로또 구입**
- 입력된 금액이 발행할 수 있는 수량을 계산하고 저장한다.
- 발행 가능한 수량은 로또 구입 금액을 나눈 값이 된다.
- 남는 금액이 존재할 경우 `IllegalArgumentException`를 발생시킨다.

**로또 생성**
- 로또 번호의 범위는 1 ~ 30 까지 이며, 중복되지 않는 숫자 6개로 구성된다.
- 생성된 로또 번호는 오름차순으로 정렬된다.

**로또 발행**
- 발행 가능한 수량 만큼 로또를 생성을 요청한다.

**당첨 번호 생성**
- 당첨 번호와 보너스 번호를 입력 받는다.
- 당첨 번호의 개수와 중복 여부를 검사한다. 예외 시 `IllegalArgumentException`를 발생시킨다.
- 당첨 번화 보너스 번호의 중복 여부를 검사한다.
- 당첨 번호를 생성하고, 오름차순으로 정렬한다.

**당첨 조회**
- 당첨 로또와 발행한 로또에서 일치하는 숫자의 개수를 리턴한다.
- 당첨 번호와 로또 번호를 비교하여 등수를 결정한다.
